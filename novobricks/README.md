# NovoBricks — Сайт производителя кирпича и плитки

## Структура проекта

```
novobricks/
├── index.html          # Главная страница (лендинг)
├── brick-quiz.html     # Квиз по кирпичу (8 вопросов + форма)
├── tile-quiz.html      # Квиз по плитке (8 вопросов + форма)
├── css/
│   └── style.css       # Все стили (дизайн-токены, компоненты, адаптив)
├── js/
│   ├── main.js         # Хедер, мобильное меню, анимации, форма
│   └── quiz.js         # Логика квизов, данные вопросов, отправка заявки
├── images/             # Сюда добавляйте изображения
└── README.md
```

## Изображения

Добавьте в папку `images/` следующие файлы:

| Файл             | Где используется              |
|------------------|-------------------------------|
| `hero-bg.jpg`    | Фон героя на главной          |
| `factory.jpg`    | Фото производства в разделе «О компании» |
| `brick.jpg`      | Превью карточки кирпича       |
| `tile.jpg`       | Превью карточки плитки        |

До добавления изображений блоки отображаются с цветовыми заглушками.

## Квиз — интеграция с CRM

В файле `js/quiz.js` в функции `leadForm` найдите комментарий:

```js
// TODO: replace with fetch('/api/quiz', { method: 'POST', body: JSON.stringify(payload) })
```

Замените на реальный запрос к вашему серверу или CRM (AmoCRM, Bitrix24, Telegram-бот и т.д.).

Объект `payload` содержит:
- `quizType` — тип квиза (`brick` или `tile`)
- `answers` — массив с вопросами и выбранными ответами
- `name`, `phone`, `email`, `comment` — данные из формы

## Запуск

Просто откройте `index.html` в браузере — никакой сборки не требуется.

Если нужен локальный сервер (для корректной работы шрифтов и путей):
```bash
npx serve .
# или
python -m http.server 3000
```
